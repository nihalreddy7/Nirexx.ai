# Nirexx.AI - World's Smartest AI-Powered Wearable

**The Future of Personal Athletic Intelligence**

A world-class, futuristic AI website for Nirexx.AI—an AI-powered next-generation wearable smartwatch designed to feel premium than Apple, Tesla, WHOOP, Oura, OpenAI, and Nothing.

## 🚀 Project Status

This is a **work in progress** built incrementally. Each section is being developed and pushed in stages.

## 🛠 Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion, GSAP
- **3D Graphics:** Three.js, React Three Fiber, Drei
- **Typography:** Inter, Space Grotesk

- ✨ Ultra-minimal luxury design language

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/nihalreddy7/Nirexx.ai.git
cd Nirexx.ai

# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Build

```bash
# Production build
npm run build

# Start production server
npm run start

# Export static HTML (for GitHub Pages)
npm run export
```
